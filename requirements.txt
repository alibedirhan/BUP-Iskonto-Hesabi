# KURULUM TALİMATLARI
# ===================

## 1. Python Kurulumu
# Python 3.8 veya üzeri gereklidir
# https://python.org adresinden indirin

## 2. Kütüphane Kurulumu
# Terminal/Command Prompt'ta şu komutu çalıştırın:
pip install -r requirements.txt

# Veya tek tek:
pip install customtkinter pandas openpyxl fpdf2 pdfplumber Pillow

## 3. Dosya Yapısı
# Tüm dosyaları aynı klasöre yerleştirin:
# - main.py
# - ui_components.py
# - pdf_processor.py
# - export_manager.py
# - requirements.txt

## 4. Çalıştırma
python main.py
